
##Wymagania
- python 3.9+
- make


## Uruchamianie
Jeśli chcesz zanstalować użyj 
```
make install
```
Jeśli chcesz uruchomić uzyj 
```
make run
```

Jeśli chcesz sprawdzic jakośc kodu uzyj 
```
make test
```
